Clean YouTube Application

This is a simple youtube playlist application. In this application, user can save various youtube playlist and chronologically play them here. During watching video, user can also take notes. This application will help user from being diverted at the time of learnig what as usual happen while learnig from youtube.

Some interesting and cool features of this application are -

  • User can add new playlist with a playlist id
  • User can add their favourite playlist videos to favourite playlists
  • User can play recently played playlist video from recently playlists
  • User can refresh their saved playlists
  • Playlists will be auto refreshed after 24 hours to bring newly uploaded videos
  • User can take notes while watching video
    These are some awesome and cool features and functionalities of this application.
